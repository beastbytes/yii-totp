# BeastBytes Yii TOTP
BeastBytes Yii TOTP is a Yii3 package that provides a TOTP (Time-based One-Time Password) implementation.

## Requirements
* PHP 8.1 or higher.

## Installation
```php
composer require beastbytes/yii-totp
```
or add the following to the 'require' section composer.json:
```json
"beastbytes/yii-totp": "*"
```
